# nao22c.github.io

Personal GitHub Pages site for nao22c iOS apps.

## Pages

- `/` — App portfolio (personal page)
- `/log_calendar/` — Log Calendar support page
- `/log_calendar/privacy-policy.html` — Log Calendar privacy policy

## Setup

1. Create a GitHub repository named `nao22c.github.io`
2. Push this directory as the repository root
3. Enable GitHub Pages from repository Settings → Pages → Source: Deploy from branch `main`

The site will be available at `https://nao22c.github.io/`
